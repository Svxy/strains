# Strain List

### Strain List is a web app used to manage and display a list of cannabis strains I've tried with detailed effect ratings. The app provides an intuitive interface to view strains, filter them based on their ratings, and explore in-depth metrics for various effects.

<br>

## Key Features

- Filter Strains: Easily filter strains by overall rating or specific effects.
- Detailed Ratings: View comprehensive ratings for each strain’s effects like relaxation, pain relief, and more.
- User-Friendly Design: A dark-themed, Bootstrap-based design for enhanced readability and accessibility.

## Usage

- View Strains: See a list of cannabis strains with their overall ratings.
- Filter Options: Choose to view strains with good or bad effects.
- Detailed Effects: Expand sections to see detailed effect ratings for each strain.
- For more information, visit the live site at https://strains.tnyavnto.com.
